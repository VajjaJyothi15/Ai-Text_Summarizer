# Ai-Text_Summarizer

# Overview

An AI-powered web application that generates concise summaries from long text documents using Transformer-based Natural Language Processing (NLP).

# Features
Summarizes long text efficiently
Supports large text inputs through chunk-based processing
Transformer-based abstractive summarization
Accuracy evaluation using TF-IDF and Cosine Similarity
Simple and responsive web interface
# Technologies Used
Python
Flask
Transformers (BART)
Scikit-learn
HTML & CSS
# Run the Project
pip install flask transformers torch scikit-learn
python app.py
